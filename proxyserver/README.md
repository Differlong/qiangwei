代理服务器相关配置和接口
========================
---
##服务端环境
* 1.一台境外VPS(当然如果你想通过代理访问国内)
* 2.代理服务器Squid
* 3.其它环境Mysql+PHP

##接口配置
telanx/config/config.php配置:

    <?php
        // 配置数据库
        $db_config = array(
            'host' => 'localhost',
            'user' => 'root',
            'pwd' => 'toor'
        );

        // 此处配置访问插件服务器白名单列表
        // 如果插件服务器与代理服务器在同一台机器上，则应该加入::1和127.0.0.1
        $whiteList = array('::1','127.0.0.1');
